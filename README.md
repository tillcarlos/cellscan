# Cellscan
A rails engine to view CSV files in a table.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "cellscan"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install cellscan
```
